# Sugarcss

<p class="typo-lead">
  Sugarcss is a [lightningcss](https://lightningcss.dev/) plugin that gives you
  access to a lot of advances features like color management/modifiers, easy
  responsive breakpoints declaration and usage, spaces management and a lot more
  to discover.
</p>

## Installation

First, install the @lotsof|sugarcss package like so:

```bash
npm i @lotsof/sugarcss
```

## Using vith Vite

To use lightningcss with Vite, first install the vite package like so:

```bash
npm i vite -D
```

Then, create a `vite.config.js` file at the root of your project with the following content:

```js
import { sugarize } from '@lotsof/sugarcss';

export default {
  css: {
    transformer: 'lightningcss',
    lightningcss: sugarize({
      minify: true,
    }),
  },
  build: {
    cssMinify: ‘lightningcss’,
  },
};
```
