# s-container
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description">
<p>This rule allows you to make the element a container easily.
The container width if relative to the <code>--s-container-max-width</code>
and the <code>--s-container-side-padding</code> variables</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">container</span><span class="_default">default</span> <span class="_type">String</span>
<p class="_description">The container you want to apply to the container&period; Can be any registered container</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   // min-width max-width side-padding
   --s-container-default: 320px 1200px 20px;
   --s-container-full: 0 100% 20px;
}

.my-element {
     @s-container();
     @s-container(full);
}
```
