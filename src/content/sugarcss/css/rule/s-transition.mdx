# s-transition
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description">
<p>This declaration allows you to apply a registered transition easily.
To be able to use this, you need to register at least 1 transition like so:</p>
<ul>
<li><code>--s-transition-{name}: all .3s ease-in-out;</code></li>
</ul>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">name<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The transition name you want to apply</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-transition-slow: all .3s ease-in-out;
}

.my-element {
     @s-transition(slow);
}
```
