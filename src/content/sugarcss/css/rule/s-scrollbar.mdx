# s-scrollbar
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description">
This rule allows you to style the scrollbar of an element or to hide it completely.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">sizeOrHide<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The size of the scrollbar or &quot;hide&quot; to hide it</p>
</li>
<li class="_item">
<span class="_name">thumbColor<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The thumb color</p>
</li>
<li class="_item">
<span class="_name">trackColor<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The track color</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
.my-element {
     @s-scrollbar(10px, s-color(accent), s-color(accent, --darken 20));
     @s-scrollbar(hide);
}
```
