# s-grid
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description">
<p>This rule allows you to apply a grid layout easily.
You can either apply a registered grid, or directly
pass the grid layout you want to apply.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">nameOrLayout<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The grid name you want to apply or directly the grid layout like &grave;1 1 2 &lowbar; 3 3 3&grave;</p>
</li>
<li class="_item">
<span class="_name">gap</span><span class="_default">0</span> <span class="_type">Number</span>
<p class="_description">The gap you want to apply between each grid cell</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-grid-default: 1 1 2 _ 3 3 3;
   --s-grid-2cols: 1 2;
}

.my-element {
   @s-grid(2cols);

  // or with an inline layout
  @s-grid('1 1 2 _ 3 3 3', 20px);
}
```
