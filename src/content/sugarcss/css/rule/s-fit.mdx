# s-fit
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description">
<p>This rule allows you to make the element fit the parent element easily.
You can choose the <code>position</code> you want as well as if you want to <code>center</code> the element
using <code>left: 50%</code>, <code>top: 50%</code> and <code>transform: translate(-50%, -50%)</code>.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">position</span><span class="_default">absolute</span> <span class="_type">String</span>
<p class="_description">The position you want to apply to the element&period; Can be &grave;absolute&grave;&comma; &grave;abs&grave;&comma; &grave;relative&grave;&comma; &grave;rel&grave;&comma; &grave;fixed&grave; or &grave;fix&grave;</p>
</li>
<li class="_item">
<span class="_name">center</span><span class="_default">false</span> <span class="_type">Boolean</span>
<p class="_description">If you want to center the element with &grave;left&colon; 50&percnt;&grave;&comma; &grave;top&colon; 50&percnt;&grave; and &grave;transform&colon; translate&lpar;-50&percnt;&comma; -50&percnt;&rpar;&grave;</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
.my-element {
     @s-fit();
     @s-fit(fixed);
     @s-fit(abs, true);
}
```
