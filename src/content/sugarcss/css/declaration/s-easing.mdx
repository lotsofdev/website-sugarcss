# s-easing
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.declaration</div>
<div class="_description">
<p>This variable allows you to declare an easing function easily and use it in your css.
You can declare as many easings as you want.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">color<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The color value to declare</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-easing-default: cubic-bezier(0.745, 0, 0.18, 1);
}

.my-element {
   transition: all 0.3s s-easing(default);
}
```
