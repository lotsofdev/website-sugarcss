# s-font
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
<p>This variable allows you to declare a font easily and use it in your css.
You can declare as many fonts as you want.</p>

<div class="_examples">
## Example
```css
:root {
   --s-font-default: s-font-family(sans) 16px 26px;
   --s-font-lead: s-font-family(sans) 26px 40px;
   --s-font-title: VT323 s-size(100) s-size(100) 300;
   --s-font-code: s-font-family(code) 16px 26px 300;
}

.my-element {
   font: s-font(default);
}
```
