# s-container
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
The container variables allows you to specify some container related variables like the max-width, side padding, etc...
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">minWidth<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The min width of the container</p>
</li>
<li class="_item">
<span class="_name">maxWidth<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The max width of the container</p>
</li>
<li class="_item">
<span class="_name">sidePadding<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The side padding of the container</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   // min-width max-width side-padding
   --s-container-default: 320px 1200px 20px;
}

.my-element {
     @s-container();
}
```
