# s-grid
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.declaration</div>
<div class="_description">
<p>This variable allows you to register some grid layouts that you can use in your css.
You can register as many grid as you want like in the example below.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">layout<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The grid layout to register like &grave;1 1 2 &lowbar; 3 3 3&grave;</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-grid-default: 1 1 2 _ 3 3 3;
   --s-grid-2cols: 1 2;
}

.my-element {
   @s-grid(default);
}
```
