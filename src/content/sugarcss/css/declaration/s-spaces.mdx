# s-spaces
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.declaration</div>
<div class="_description">
<p>This variable allows you to declare the spaces values to use in your css.
You can declare as many spaces as you want.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">min<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The easing value for the min space</p>
</li>
<li class="_item">
<span class="_name">max<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The easing value for the max space</p>
</li>
<li class="_item">
<span class="_name">easing</span><span class="_default">linear</span> <span class="_type">String</span>
<p class="_description">The easing value for the space</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
     --s-spaces: 0 80px linear;
}

.my-element {
   padding: s-space(10); // 8px
   padding: s-space(20); // 16px
   padding: s-space(100); // 80px
}
```
