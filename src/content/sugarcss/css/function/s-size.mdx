# s-size
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_description">
<p>This function allows you to apply a size depending on the
min, max and easing function declared using the <code>--s-sizes</code> variable.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">size<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
<p class="_description">The size you want to apply between 0 and 100</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
     --s-sizes: 0px 80px linear;
}

.my-element {
   font-size: s-size(10); // 8px
   font-size: s-size(20); // 16px
   font-size: s-size(100); // 80px
}
```
