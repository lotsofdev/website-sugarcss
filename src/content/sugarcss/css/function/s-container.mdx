# s-container
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_description">
<p>Allow to get some container related properties like:</p>
<ul>
<li><code>width</code>: The calculated container width relative to the <code>--s-container-max-width</code> and the <code>--s-container-side-padding</code> variables</li>
<li><code>minWidth</code> (<code>min-width</code>): The min width of the container</li>
<li><code>maxWidth</code> (<code>max-width</code>): The max width of the container</li>
<li><code>sidePadding</code> (<code>side-padding</code>): The side padding of the container</li>
</ul>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">prop<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The container property wanted</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   // min-width max-width side-padding
   --s-container-default: 320px 1200px 20px;
}

.my-element {
   width: s-container(width);

   min-width: s-container(minWidth);
   max-width: s-container(maxWidth);
   padding-left: s-container(sidePadding);
}
```
