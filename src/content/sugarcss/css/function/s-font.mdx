# s-font
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_description">
<p>Apply a font from the fonts registered ones.
The supported fonts properties are:</p>
<ul>
<li><code>family</code>: The font family like &quot;sans-serif&quot;, &quot;serif&quot;, etc...</li>
<li><code>size</code>: The font size like &quot;1em&quot;, &quot;12px&quot;, etc...</li>
<li><code>lineHeight</code>: The font line-height like &quot;1em&quot;, &quot;12px&quot;, etc...</li>
<li><code>weight</code>: The font weight like &quot;normal&quot;, &quot;bold&quot;, etc...</li>
<li><code>style</code>: The font style like &quot;normal&quot;, &quot;italic&quot;, etc...</li>
<li><code>variant</code>: The font variant like &quot;normal&quot;, &quot;small-caps&quot;, etc...</li>
<li><code>stretch</code>: The font stretch like &quot;normal&quot;, &quot;condensed&quot;, etc...</li>
</ul>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">font<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The font name you want to apply from registered ones</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
  --s-font-...: "Roboto,sans-serif" %size %lineHeight %weight %style %variant %stretch;
  --s-font-family-code: "Fira Code", monospace;
  --s-font-code: s-font-family(code) 16px 26px 300;
}

.my-element {
  font: s-font(code);
}
```
