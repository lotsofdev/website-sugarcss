# s-radius
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>

<div class="_description">
This function allows you to apply a radius from the radiuses registered ones.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">radius<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The radius name you want to apply from registered ones</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-radius-default: 10px;
   --s-radius-special: 20px 10px 12px 34px;
}

.my-element {
   border-radius: s-radius();
   border-radius: s-radius(special);
}
```
