# s-color
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_description">
<p>Apply a color from the colors registered ones.
Allow to apply modifiers or shades to the color like darken, lighten, saturate, desaturate, spin, etc...
Here&#39;s the list of available modifiers:</p>
<ul>
<li><code>lighten</code>: 0-100</li>
<li><code>darken</code>: 0-100</li>
<li><code>saturate</code>: 0-100</li>
<li><code>desaturate</code>: 0-100</li>
<li><code>spin</code>: 0-360</li>
<li><code>alpha</code>: 0-1</li>
</ul>
<p>The shades are registered like so: --s-shade-placeholder: --darken 10;</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">color<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The color name you want to apply like "main", "accent", etc...</p>
</li>
<li class="_item">
<span class="_name">shadeOrModifiers<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The shade name you want to apply like "text", "placeholder", etc... or a string of modifiers like --lighten 10 --saturate 20</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-shade-test: --lighten 10;
}

.my-element {
  color: s-color(accent);
  color: s-color(accent, test);
  color: s-color(accent, --darken 20 --spin 30);
}
```
