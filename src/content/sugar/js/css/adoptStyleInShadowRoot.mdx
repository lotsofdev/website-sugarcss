## Params
1. <span class="_name">$shadowRoot<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLShadowRootElement</span>

2. <span class="_name">$context</span><span class="_default">document</span> <span class="_type">HTMLElement</span>
   - The context from which you want to adopt the styles
</div>
<div class="_return">
## Return
- <span class="_description">Return a promise fullfilled when the styles have been adopted</span><span class="_default"></span><span class="_type">Promise</span>
</div>
<div class="_examples">
## Examples
```js

await __adoptStyleInShadowRoot(myShadowRoot);
```
