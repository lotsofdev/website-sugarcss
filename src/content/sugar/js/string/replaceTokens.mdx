# replaceTokens
<div class="_namespace">@lotsof.sugar.js.string</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function takes as parameter a tokened string like "something [cool]", an object
of arguments/values and return the processed string with the tokens replaced by the arguments values.
</div>
<div class="_params">
## Params
1. <span class="_name">string<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The string to process
2. <span class="_name">argsObj<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The arguments/value object
3. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - A settings object to configure the parsing process
</div>
<div class="_return">
## Return
- <span class="_description">The processed string</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __replaceTokens } from '@lotsof/sugar/string';
__replaceTokens('hello [world]', { world: 'Coco' }); // => hello Coco
```
</div>
<div class="_settings">
## Settings
1. <span class="_name">[regexp=\[([a-zA-Z0-9-_]+)\] The regexp to use to match the tokens<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
2. <span class="_name">stripUndefined</span><span class="_default">true</span> <span class="_type">Boolean</span>
</div>
