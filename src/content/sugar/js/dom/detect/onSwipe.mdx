<div class="_params">
## Params
1. <span class="_name">elm<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLElement</span>
   - The HTMLElement on which to detect the swipe
2. <span class="_name">cb<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The function to call on swipe. The callback function has as parameter an object that containthe swipe direction like left, right, up and down
3. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">Partial<IOnSwipeSettings></span>
   - Some settings to configure your swipe detection
</div>
<div class="_examples">
## Examples
```js
import { __onSwipe } from '@coffeekraken/sugar/dom'
__onSwipe(myCoolElm, (swipe) => {
	// check the swipe direction
	if (swipe.left) {
		// do something...
	}
	// support : left, right, up, down
	// etc...
}, {
	threshold : 50
});
```
</div>
