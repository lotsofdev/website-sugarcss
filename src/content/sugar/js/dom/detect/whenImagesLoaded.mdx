# whenImagesLoaded
<div class="_namespace">@lotsof.sugar.js.dom.detect</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Detect when some images are loaded. This function take advantage of the SPromise class
and trigger an event called "img.loaded" that will be triggered on each loaded images and another called "loaded" once all the images are loaded.
See in the example bellow.
</div>
<div class="_params">
## Params
1. <span class="_name">$imgs<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array<HTMLImageElement></span>
   - An array (or nodeList) of HTMLImageElement to detect the load
</div>
<div class="_return">
## Return
- <span class="_description">A promise resolved when all images are loaded properly</span><span class="_default"></span><span class="_type">Promise</span>
</div>
<div class="_examples">
## Examples
```js
import { __whenImagesLoaded } from '@coffeekraken/sugar/dom'
__whenImagesLoaded([
	$img1, $img2, $img3
]).on('loaded', $img => {
   // do something with the loaded image
}).then(imgs => {
  // do something here
})
```
