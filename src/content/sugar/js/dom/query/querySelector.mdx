# querySelector
<div class="_namespace">@lotsof.sugar.js.dom.query</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Enhanced proxy of the Element.querySelector function that let you specify
if you want an element that is visible, or even that is in the viewport
</div>
<div class="_params">
## Params
1. <span class="_name">selector<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The css selector to search
2. <span class="_name">settings<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The settings of the query
</div>
<div class="_return">
## Return
- <span class="_description">The founded element</span><span class="_default"></span><span class="_type">HTMLElement</span>
</div>
<div class="_examples">
## Examples
```js
import { __querySelector } from '@coffeekraken/sugar/dom';
// simple query
const elm = __querySelector('.a-cool-css-selector');

// get an element that is in the viewport
const elm = __querySelector('.a-cool-css-selector', {
		inViewport : true
});
```
</div>
<div class="_settings">
## Settings
1. <span class="_name">visible</span><span class="_default"></span> <span class="_type">Boolean</span>
2. <span class="_name">inViewport</span><span class="_default"></span> <span class="_type">Boolean</span>
3. <span class="_name">rootNode</span><span class="_default">document.body</span> <span class="_type">HTMLElement</span>
