# chdir
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function allows you to change the process working directory.
If you make use of the `__spawn` sugar function, it will also change
the directory of the parent process if you call this function in a childProcess.
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The folder path you want to go to
</div>
<div class="_examples">
## Examples
```js
import { __chdir } from '@lotsof/sugar/fs';
__chdir('/my/cool/file.txt', 'Hello world');
```
