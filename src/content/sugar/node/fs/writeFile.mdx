# writeFile
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
CWrite a file. If don't exist, will be created as well as the directory structure if needed... ( (async)
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The file path to write
2. <span class="_name">data<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The data to write in the file
3. <span class="_name">options</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - options are what you'd pass to [fs.writeFile()](https://nodejs.org/api/fs.html#fs_fs_writefile_file_data_options_callback)
</div>
<div class="_return">
## Return
- <span class="_description">A promise that will be resolved when the writeFile is completed</span><span class="_default"></span><span class="_type">Promise</span>
</div>
<div class="_examples">
## Examples
```js
import { __writeFile } from '@lotsof/sugar/fs';
__writeFile('my/cool/file.txt', 'Hello World').then(() => {
   // do something on complete...
});
```
