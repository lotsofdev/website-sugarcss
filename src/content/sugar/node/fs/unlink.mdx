# unlink
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
unlinks a file or directory. The directory can have contents. If the path does not exist, silently does nothing. Like rm -rf (async)
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The file/directory path to delete
</div>
<div class="_return">
## Return
- <span class="_description">A promise that will be resolved when the unlink is completed</span><span class="_default"></span><span class="_type">Promise</span>
</div>
<div class="_examples">
## Examples
```js
import { __unlink } from '@coffeekraken/sugar/fs';
await __unlink('my/cool/file.json').then(() => {
   // do something on complete...
});
```
