# folderPath
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function returns you the folder path of the file path passed.
You can tell the function to check for file existence before getting
the folder path with the second parameter.
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The file path to get folder path from
2. <span class="_name">checkExistence</span><span class="_default">false</span> <span class="_type">Boolean</span>
   - Specify if you want to check the file existence before
</div>
<div class="_return">
## Return
- <span class="_description">The folder path or '' if not exists</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __folderPath } from '@lotsof/sugar/fs';
__folderPath('my/cool/path.js'); // => true
```
