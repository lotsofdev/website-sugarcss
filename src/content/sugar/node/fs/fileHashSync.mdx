# fileHashSync
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function allows you to get back an integrity hash for the passed file.
This mean that if a file returns the same integrity hash twice, the folder or files in it
has not been updated...
</div>
<div class="_params">
## Params
1. <span class="_name">filePath<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The folder path you want to get the hash back
2. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">IFileHashSettings</span>
   - Some settings to configure your hash generation process
</div>
<div class="_return">
## Return
- <span class="_description">The calculated folder hash</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __fileHashSync } from '@lotsof/sugar/fs';
__fileHashSync('my/cool/folder'); // => YZOrKDx9LCLd8X39PoFTflXGpRU=,
```
</div>
<div class="_settings">
## Settings
1. <span class="_name">algo</span><span class="_default">sha356</span> <span class="_type">String</span>
2. <span class="_name">digest</span><span class="_default">base64</span> <span class="_type">BinaryToTextEncoding</span>
