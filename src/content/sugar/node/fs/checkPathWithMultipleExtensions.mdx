# checkPathWithMultipleExtensions

<div class="_metas">

<div class="_description">
This function take a path and some extensions to check if a file
exists with one of these particular extensions.
If a file exists, the function return the path with the first extensions that matches
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The file path you want to check. With or without an extension
2. <span class="_name">extensions<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array<String></span>
   - The extensions (without the dot) you want to check
</div>
<div class="_return">
## Return
- <span class="_description">The first valid path founded, or undefined</span><span class="_default"></span><span class="_type">String|undefined</span>
</div>
<div class="_examples">
## Examples
```js
import { __checkPathWithMultipleExtensions } from '@lotsof/sugar/fs';
__checkPathWithMultipleExtensions('/my/cool/file.txt', ['txt','js','css']);
```
