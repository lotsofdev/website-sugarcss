# prependToFileSync
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function allows you to simply append a string to a file.
</div>
<div class="_params">
## Params
1. <span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The file path you want to check. With or without an extension
2. <span class="_name">content<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The content to add to the file
</div>
<div class="_examples">
## Examples
```js
import { __prependToFileSync } from '@lotsof/sugar/fs';
__prependToFileSync('/my/cool/file.txt', 'Hello world');
```
