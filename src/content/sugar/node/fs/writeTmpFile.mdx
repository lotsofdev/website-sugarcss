# writeTmpFile
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_description">
Create a temporary file on the disk with the passed content and returns the path
to it.
</div>
<div class="_params">
## Params
1. <span class="_name">data<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Any</span>
   - The data to write in the file
2. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">IWriteTmpFileSettings</span>
   - Some settings to customize your temp file creation
</div>
<div class="_return">
## Return
- <span class="_description">A promise that will be resolved when the writeTmpFile is completed with the path to it</span><span class="_default"></span><span class="_type">Promise<String></span>
</div>
<div class="_examples">
## Examples
```js
import { __writeTmpFile } from '@lotsof/sugar/fs';
__writeTmpFile('Hello World').then((path) => {
   // do something on complete...
});
```
</div>
