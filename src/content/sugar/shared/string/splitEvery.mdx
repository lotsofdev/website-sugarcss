# splitEvery
<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Split a string every n chars either by taking care of not spliting the words, or by simply spliting without any attention to that...
</div>
<div class="_params">
## Params
1. <span class="_name">text<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The text to split
2. <span class="_name">every<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - How many characters to split the text
3. <span class="_name">settings</span><span class="_default">{('esm'|'cjs')[]}</span> <span class="_type">Partial<ISplitEverySettings</span>
   - Some settings to configure your split process
</div>
<div class="_return">
## Return
- <span class="_description">An array of the splited text parts</span><span class="_default"></span><span class="_type">Array</span>
</div>
<div class="_examples">
## Examples
```js
import { __splitEvery } from '@lotsof/sugar/string';
__splitEvery('Hello World', 2, {
   splitWords: true
}); // => ['He','ll','o ','Wo','rl','d']
```
</div>
<div class="_settings">
## Settings
1. <span class="_name">splitWords</span><span class="_default">true</span> <span class="_type">Boolean</span>
