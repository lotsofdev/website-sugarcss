# countLineChars

<div class="_metas">

<div class="_description">
Count how many characters their is in the passed line.
This function will exclude the characters like the html tags like <red>, etc...
</div>
<div class="_params">
## Params
1. <span class="_name">line<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The line to count
2. <span class="_name">count</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - Specify what you want to count outside of the normal characters of yourse. Here's the list of available options:
</div>
<div class="_return">
## Return
- <span class="_description">How many characters their is in the line</span><span class="_default"></span><span class="_type">Number</span>
</div>
<div class="_examples">
## Examples
```js
import { __countLineChars } from '@lotsof/sugar/string';
 __countLineChars('Hello <red>World</red>'); // 11
```
