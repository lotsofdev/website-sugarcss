# replaceChunks
<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Replace the passed "chunks" with the result of the passed replacer function.
The replacer function take as argument the token value
</div>
<div class="_params">
## Params
1. <span class="_name">string<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The string to process
2. <span class="_name">chunks<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String[]</span>
   - The chunks to replace
3. <span class="_name">replacer<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The replacer function that need to return a string with which you want to replace the token
</div>
<div class="_return">
## Return
- <span class="_description">The new string with replaced chunks</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __replaceChunks } from '@coffeekraken/sugar/string'
__replaceChunks('hello world', ['or'], (token) => {
     return `<span>${token}</span>;
}) // Hello w<span>or</span>ld
```
