# extractSame
<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function return you what has been find the same in the two passed string.
It will return you either an array of same string parts or a simple string
representing the first same part found.
</div>
<div class="_params">
## Params
1. <span class="_name">string1<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The string 1 to compare
2. <span class="_name">string2<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The string 2 to compare
3. <span class="_name">multiple</span><span class="_default">false</span> <span class="_type">Boolean</span>
   - Specify if you want to get back multiple same string if exists as an array
</div>
<div class="_return">
## Return
- <span class="_description">The same string part(s) found</span><span class="_default"></span><span class="_type">String|Array</span>
</div>
<div class="_examples">
## Examples
```js
import { __extractSame } from '@lotsof/sugar/string';
__extractSame('Hello world', 'Hello plop'); // => 'Hello '
```
