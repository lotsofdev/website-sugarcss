# sprintf
<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Javascript implementation of the sprintf php function.
>For more infos, check [this github repository](https://github.com/alexei/sprintf.js)
</div>
<div class="_params">
## Params
1. <span class="_name">format<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The format of the string wanted as output
2. <span class="_name">...replacements<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Mixed</span>
   - The replacement tokens to apply to the string
</div>
<div class="_return">
## Return
- <span class="_description">The processed string</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __sprintf } from '@coffeekraken/sugar/string'
__sprintf('Hello %s', 'world') // Hello World
const user = { name: 'Dolly' }
__sprintf('Hello %(name)s', user) // Hello Dolly
```
