# clone
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_description">
This function allows you to clone an object either at 1 level, or deeply.
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to copy
2. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - Specify some settings to configure your clone process
</div>
<div class="_return">
## Return
- <span class="_description">The cloned object</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __clone } from '@lotsof/sugar/object';
__clone({
   hello: 'world'
});
```
</div>
