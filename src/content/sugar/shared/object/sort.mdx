# sort
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Sort an object properties the same way as the Array.sort do it.
The "a" and "b" argument passed to your sort function will have these properties:
- key: The key of the object
- value: The actual value of the object property
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to sort
2. <span class="_name">sort</span><span class="_default"></span> <span class="_type">Function</span>
   - The sort function to use. If not specified, will sort the items by key alphabetically
</div>
<div class="_return">
## Return
- <span class="_description">The sorted object</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __sort } from '@lotsof/sugar/object';
__sort({
   coco: { weight: 10 },
   lolo: { weight: 2 },
   plop: { weight: 5 }
}, (a, b) => {
  return a.value.weight - b.value.weight;
});
// {
//   lolo: { weight: 2 },
//   plop: { weight: 5 },
//   coco: { weight: 10 }
// }
```
</div>
