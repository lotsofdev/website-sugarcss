<div class="_metas">

Convert class instances to plain JSON object
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Any</span>
   - The object to convert
</div>
<div class="_return">
## Return
- <span class="_description">The converted object</span><span class="_default"></span><span class="_type">Any</span>
</div>
<div class="_examples">
## Examples
```js
import { __toJson } from '@lotsof/sugar/object';
class MyClass {
     hello = 'world';
     something() {}
}
__toJson(new MyClass()); // => { hello: 'world' }
```
