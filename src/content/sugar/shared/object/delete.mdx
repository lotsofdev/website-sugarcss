# delete
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Delete an object/array property by passing a dotpath to the element you want to remove.
</div>
<div class="_params">
## Params
1. <span class="_name">obj<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object in which to set the value
2. <span class="_name">dotpath<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String|String[]</span>
   - The dotted object/array path to delete
</div>
<div class="_examples">
## Examples
```js
import { __delete } from '@lotsof/sugar/object';
__delete(myObject, 'my.cool.value');
__delete(myObject, 'my.cool.value.0');
```
