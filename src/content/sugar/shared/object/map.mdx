# map
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This is the same function as the "Array.map" but for objects. It will iterate over all the properties
of the passed object and pass the value to your process function. It will then save the property
with your processed value
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to process
2. <span class="_name">processor<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The processor function that will take as parameters the current property value and the property name
</div>
<div class="_return">
## Return
- <span class="_description">The processed object</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __map } from '@lotsof/sugar/object';
const myObject = {
   hello: 'world',
   cat: 'Nelson'
};
__map(myObject, ({value, prop}) => {
   return prop === 'hello' ? 'universe' : value;
});
{
   hello: 'universe',
   cat: 'Nelson'
}
```
