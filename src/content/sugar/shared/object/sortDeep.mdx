# sortDeep
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_description">
Sort an object properties the same way as the Array.sort do it but will do it recusrively to sort the object deeply.
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to sort
2. <span class="_name">sort<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The sort function to use
</div>
<div class="_return">
## Return
- <span class="_description">The sorted object</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __sortDeep } from '@lotsof/sugar/object';
__sortDeep({
   lolo: { weight: 2 },
   coco: { weight: 10 },
   plop: { weight: 5 },
   aha: {
     hello: 'world',
     coco: 'plop'
   }
}, (a, b) => {
   return a.key.localeCompare(b.key);
});
// {
//   aha: {
//      coco: 'plop',
//      hello: 'world'
//   }
//   coco: { weight: 10 }
//   lolo: { weight: 2 },
//   plop: { weight: 5 },
// }
```
