# deepize
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

</div>
<div class="_description">
This function simply take an object like this one:
{
'something.cool': 'hello'
}
and convert it to something like this:
{
something: {
cool: 'hello'
}
}
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to convert
</div>
<div class="_return">
## Return
- <span class="_description">The converted object</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __deepize } from '@lotsof/sugar/object';
__deepize ({ 'something.cool': 'hello' }); // => { something: { cool: 'hello' } }
```
