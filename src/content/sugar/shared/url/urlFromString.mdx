# urlFromString
<div class="_namespace">@lotsof.sugar.shared.url</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Simple function that take a string as parameter and returns you a valid
url ready one
</div>
<div class="_examples">
## Examples
```js
import { __urlFromString } from '@lotsof/sugar/url';
__urlFromString('Sir James Paul McCartney MBE is an English singer-songwriter');
// sir-james-paul-mc-cartney-mbe-is-an-english-singer-songwriter
```
