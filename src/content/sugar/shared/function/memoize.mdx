# memoize
<div class="_namespace">@lotsof.sugar.shared.function</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function can be used to memorize a function call result in order to save memory.
This is just a "proxy" of the "memoizee" package.
</div>
<div class="_params">
## Params
1. <span class="_name">fn<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The function to memoize his result
</div>
<div class="_return">
## Return
- <span class="_description">The memoized function result</span><span class="_default"></span><span class="_type">Any</span>
</div>
<div class="_examples">
## Examples
```js
import { __memoize } from '@lotsof/sugar/function';
const fn =  __memoize(function(text) => {
   return `Hello ${text}`;
});
const result = fn('world'); // first execution. no cache
const result1 = fn('plop'); // first execution with this argument, no cache
const result2 = fn('world'); // taken from cache
```
