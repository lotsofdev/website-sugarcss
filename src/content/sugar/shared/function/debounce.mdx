# debounce
<div class="_namespace">@lotsof.sugar.shared.function</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This utils function allows you to make sure that a function that will normally be called
several times, for example during a scroll event, to be called only once after
the delay passed
</div>
<div class="_params">
## Params
1. <span class="_name">delay<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - A delay in ms to wait between two function calls
2. <span class="_name">fn<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
   - The function to debounce
</div>
<div class="_examples">
## Examples
```js
import { __debounce } from '@lotsof/sugar/function';
const myDebouncedFn = __debounce(1000, () => {
		// my function content that will be
		// executed only once after the 1 second delay
});

document.addEventListener('scroll', (e) => {
		// call my debounced function
		myDebouncedFn();
});
```
