# rgba2hex
<div class="_namespace">@lotsof.sugar.shared.color</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
RGBA to HEX
</div>
<div class="_params">
## Params
1. <span class="_name">r<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number|String|Object</span>
   - The red value between 0-255 or an object representing r, g, b, a or a string representing the rgba(...) color
2. <span class="_name">g<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - The green value between 0-255
3. <span class="_name">b<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - The blue value between 0-255
4. <span class="_name">a<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - The alpha value between 0-100|0-1
</div>
<div class="_return">
## Return
- <span class="_description">The hex string representation like #ff004f</span><span class="_default"></span><span class="_type">string</span>
</div>
<div class="_examples">
## Examples
```js
import { __rgbaToHex } from '@lotsof/sugar/color';
__rgbaToHex(10,20,30,10);
```
