# parseAuthorString
<div class="_namespace">@lotsof.sugar.shared.package</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function simply take an author string like "Olivier Bossel <olivier.bossel@gmail.com> (https://coffeekraken.io)" and
transform it into a plain object with these properties: name, email and url
</div>
<div class="_params">
## Params
1. <span class="_name">string<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The string to parse
</div>
<div class="_return">
## Return
- <span class="_description">The plain object version of the string</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __parseAuthorString } from '@lotsof/sugar/package';
 __parseAuthorString("Olivier Bossel <olivier.bossel@gmail.com> (https://coffeekraken.io)")
// => {
  "name": "Olivier Bossel",
  "email": "olivier.bossel@gmail.com",
  "url": "https://olivierbossel.com"
}
```
