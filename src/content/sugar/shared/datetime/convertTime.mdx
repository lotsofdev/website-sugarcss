# convertTime
<div class="_namespace">@lotsof.sugar.shared.datetime</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function allows you to convert time like seconds, ms, hours, minutes, etc... from one format to another
</div>
<div class="_params">
## Params
1. <span class="_name">from<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String|Number</span>
   - The value to start from like "10s", "20ms", "2h", etc...
2. <span class="_name">to</span><span class="_default">ms</span> <span class="_type">String</span>
   - The format you want to get back
</div>
<div class="_return">
## Return
- <span class="_description">The converted value</span><span class="_default"></span><span class="_type">Number</span>
</div>
<div class="_examples">
## Examples
```js
import { __convertTime } from '@lotsof/sugar/datetime';
__convertTime('10s', 'ms'); // => 10000
```
