# splitEvery

<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
Split an array every N items
</div>
<div class="_params">
## Params
1. <span class="_name">array<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array</span>
   - The array to split
2. <span class="_name">every<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
   - Every how many items to split the array
</div>
<div class="_return">
## Return
- <span class="_description">An array of arrays splited</span><span class="_default"></span><span class="_type">Array</span>
</div>
<div class="_examples">
## Examples
```js
import { __splitEvery } from '@lotsof/sugar/array';
__splitEvery([1,2,3,4,5,6,7,8,9], 3);
// [[1,2,3],[4,5,6],[7,8,9]]
```
