# intersection

<div class="_description">
This function take two arrays and return all the items that are the same.
You can specify if you want only comparing by reference for object, etc... or if
you want to transform items in hashes and compare that instead.
</div>
<div class="_params">
## Params
1. <span class="_name">...arrays<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array</span>
   - The arrays you want to compare
2. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - Some settings to refine your comparaison process
</div>
<div class="_return">
## Return
- <span class="_description">An array containing all the items that are present in all the passed arrays</span><span class="_default"></span><span class="_type">Array</span>
</div>
<div class="_examples">
## Examples
```js
import { __intersection } from '@lotsof/sugar/array';
__intersection([1,2,3,4], [1,3,5]); // => [1,3]
```
</div>
