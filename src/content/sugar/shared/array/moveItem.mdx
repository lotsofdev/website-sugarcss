# moveItem
<div class="_namespace">@lotsof.sugar.shared.array</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
</div>
<div class="_description">
This function allows you to move an item to another place in the array
</div>
<div class="_params">
## Params
1. <span class="_name">array<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array</span>
   - The array to process
2. <span class="_name">from<span class="_required">*</span></span><span class="_default"></span> <span class="_type">any|integer</span>
   - Either the element itself that you want to move, or the index of the element you want to move
3. <span class="_name">to<span class="_required">*</span></span><span class="_default"></span> <span class="_type">any|integer</span>
   - Either the element itself where you want to move the first one, or the index of the element you want to move your element to
</div>
<div class="_return">
## Return
- <span class="_description">The processed array</span><span class="_default"></span><span class="_type">Array</span>
</div>
<div class="_examples">
## Examples
```js
import { __moveItem } from '@lotsof/sugar/array';
const myArray = ['hello','world','something'];
__moveItem(myArray, 0, 1); // ['world','hello','something']
```
