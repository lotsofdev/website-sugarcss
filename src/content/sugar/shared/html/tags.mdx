# tags
<div class="_namespace">@lotsof.sugar.shared.html</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
An object of tags to replace which have as value the replacement function that take the tag name, the tag content and must return the replacement content
</div>
<div class="_return">
## Return
- <span class="_description">The new text</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __replaceTags } from '@lotsof/sugar/html';
 __replaceTags('<span>Hello</span> world', {
   span: (tag, content) => `<div>${content}</div>`; // => <div>Hello</div> world
});
```
