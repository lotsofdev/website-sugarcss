# expandPleasantCssClassnames
<div class="_namespace">@lotsof.sugar.shared.html</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function allows you to convert "colon" classnames like "s-something:cool @desktop something"
to comprehensive classnames for css like "s-something s-something-cool something_desktop", etc...
</div>
<div class="_params">
## Params
1. <span class="_name">html<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The HTML to process. It can be actually any string values like .vue file, etc...
</div>
<div class="_return">
## Return
- <span class="_description">The processed string with converted classnames</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __expandPleasantCssClassnames } from '@lotsof/sugar/html';
__expandPleasantCssClassnames('...');
```
