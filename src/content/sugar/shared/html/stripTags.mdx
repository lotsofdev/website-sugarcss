# stripTags
<div class="_namespace">@lotsof.sugar.shared.html</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_description">
Strip tags of an html string.
This is a simple wrapper of the nice "striptags" package that you can find here: https://www.npmjs.com/package/striptags
</div>
<div class="_params">
## Params
1. <span class="_name">html<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The html string to process
2. <span class="_name">allowedTags<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String[]</span>
   - The tags that are allowed like ['a','p','h1','h2']...
3. <span class="_name">tagReplacement<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - A string with which you want to replace the tags
</div>
<div class="_return">
## Return
- <span class="_description">The processed string without tags</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __stripTags } from '@coffeekraken/sugar/html'
__stripTags('<p><span>Hello</span> world</p>', '<span>') // <span>Hello</span> world
```
