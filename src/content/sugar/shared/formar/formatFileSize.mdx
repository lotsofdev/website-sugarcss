# formatFileSize
<div class="_namespace">@lotsof.sugar.shared.formar</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Transform into human readable string a file size from a number (float or integer) or string.
This function use the wonderfull "filesize" npm package under the houd.
</div>
<div class="_params">
## Params
1. <span class="_name">size<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number|String</span>
   - The size to transform in bytes
2. <span class="_name">settings</span><span class="_default">[object Object]</span> <span class="_type">Object</span>
   - The "filesize" settings to customize the output. See [filesize](https://www.npmjs.com/package/filesize) settings
</div>
<div class="_return">
## Return
- <span class="_description">The human readable version of the passed size</span><span class="_default"></span><span class="_type">String</span>
</div>
<div class="_examples">
## Examples
```js
import { __formatFilesize } from '@lotsof/sugar/fs';
__formatFilesize(163931); // => 326.86 KB
```
</div>
