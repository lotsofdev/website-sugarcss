# toPlainObject
<div class="_namespace">@lotsof.sugar.shared.class</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
This function take a instance as parameter and return a plain object of it
</div>
<div class="_params">
## Params
1. <span class="_name">instance<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Mixed</span>
   - Any class instance to transform into a plain object
</div>
<div class="_return">
## Return
- <span class="_description">A plain object version of the the class instance</span><span class="_default"></span><span class="_type">Object</span>
</div>
<div class="_examples">
## Examples
```js
import { __toPlainObject } from '@lotsof/sugar/class';
class Coco {
   constructor() {
     this.hello = 'world';
   }
}
__toPlainObject(new Coco()); // => { hello: 'world' }
```
