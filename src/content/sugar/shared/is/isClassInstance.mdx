# isClassInstance
<div class="_namespace">@lotsof.sugar.shared.is</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Check if the passed item is an object class and not a plain object.
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Any</span>
   - The object to check
</div>
<div class="_return">
## Return
- <span class="_description">true if is an custom object instance, false if not</span><span class="_default"></span><span class="_type">Boolean</span>
</div>
<div class="_examples">
## Examples
```js
import { __isClassInstance } from '@lotsof/sugar/is';
if (__isClassInstance({
     something: 'hello'
})); // => false
class MyClass {
     constructor() {}
}
__isClassInstance(new MyClass());
```
