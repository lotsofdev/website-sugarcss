# cyclic
<div class="_namespace">@lotsof.sugar.shared.is</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_description">
This function check if the passed object has circular dependencies and if so, returns where it has been found in the object
</div>
<div class="_params">
## Params
1. <span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
   - The object to check
</div>
<div class="_return">
## Return
- <span class="_description">false if all is ok, a string that tells where the circular dep has been found if not</span><span class="_default"></span><span class="_type">String|false</span>
</div>
<div class="_examples">
## Examples
```js
import { __isCyclic } from '@lotsof/sugar/is';
const obj = { hello: 'world' };
obj.cyclic = obj;
__isCyclic(obj);
```
