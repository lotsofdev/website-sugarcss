# isValidNumber
<div class="_namespace">@lotsof.sugar.shared.is</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
</div>
<div class="_description">
Check if the passed value is a valid number.
A valid number is either an integer or a number (float).
</div>
<div class="_params">
## Params
1. <span class="_name">value<span class="_required">*</span></span><span class="_default"></span> <span class="_type">any</span>
   - The value to check
</div>
<div class="_return">
## Return
- <span class="_description">true if it's in a valid number, false if not</span><span class="_default"></span><span class="_type">Boolean</span>
</div>
<div class="_examples">
## Examples
```js
import { __isValidNumber } from '@coffeekraken/sugar/is'
__isValidNumber(12) // true
__isValidNumber('he') // false
__isValidNumber(undefined) // false
__isValidNumber(NaN) // false
```
